Instale as dependências:
```
npm i
```

Inicie o servidor
```
npm start
```

Rodar o código gerado pelo TS
```
npm run builded
```

Testes Unitários
```
npm run unit-test
```

Testes de Integração
```
npm run integration-test
```

Coverage
```
npm run integration-coverage
npm run unit-coverage
```
